# Домашнее задание №1

## 1.1. Вывод чисел с помощью функции even

Напишите функцию с названием even, которая принимает на вход целое число. Она должна вернуть количество всех положительных четных чисел от 1 до этого числа включительно.

Результат работы функции even будет выводиться в `printf("%ld ", num)`, где `num` - это количество положительных четных чисел.

## 1.2. Возведение в степень с помощью функции pow

Напишите функцию с названием `pow`. Функция принимает на вход два целых числа: число и степень, в которую нужно возвести это число.

Функция возвращает целое число, которое получилось в результате возведения в степень. Если вычисление невозможно произвести, верните 0. Используйте рекурсию.
