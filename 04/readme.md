# Домашнее задание №4

## 4.1. Вывод массива с распределением количества символов строки

Напишите функцию с названием `charCount`. Функция принимает на вход строку и отдает обратно массив с распределением количества символов в этой строке.

Массив формируется следующим образом: символ приводится к его численному виду и используется как индекс этого массива. Значение элемента массива содержит значение, показывающее, сколько раз встретился этот символ.
Вызывающая функция отвечает за очистку памяти этого массива. Проверяются только печатные символы.

## 4.2. Перевод буквенных символов в верхний регистр

Напишите функцию с названием `upperCase`, которая принимает строку и видоизменяет ее таким образом, что все буквенные символы в ней переводятся в верхний регистр. Функция ничего не возвращает, библиотечные функции использовать запрещено.
