# Домашнее задание №9

## 9.1. Сложение всех элементов численных массивов

Реализуйте шаблонную функцию

```C
    template <class T>
    T sum(const T* val, size_t size);
```

Функция должна складывать все элементы численных массивов и возвращать результат.

Определите специализацию шаблона для класса `String`. Функция должна складывать строки и возвращать как результат новую структуру. Класс принимает в конструкторе строку и работает с ее копией. У класса имеется метод `data` для доступа к строке.

## 9.2. Реализация шаблонного класса Stack

Реализуйте шаблонный класс `Stack`, который представляет из себя контейнер, действующий по принципу __LIFO__.
Реализуйте методы для работы с классом - `push` и `pop`, которые позволяют добавить или удалить элемент.
Реализуйте метод `top`, который возвращает текущий элемент на вершине стека.
Реализуйте метод `size` для получения количества элементов в контейнере.
Правильно расставьте модификаторы доступа.
Если операцию провести невозможно, выбросьте исключение `std::out_of_range` из стандартной библиотеки C++. Исключение должно содержать текст с названием метода, из которого оно было выброшено.
